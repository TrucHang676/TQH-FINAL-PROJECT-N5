# Dashboard phân tích xu hướng tuyển dụng và cơ hội nghề nghiệp ngành Công nghệ thông tin tại Việt Nam

## 1. Giới thiệu

Đồ án xây dựng dashboard trực quan hóa dữ liệu nhằm phân tích thị trường tuyển dụng ngành Công nghệ thông tin tại Việt Nam. Dashboard tập trung vào nhu cầu tuyển dụng, kỹ năng được yêu cầu, mức lương, địa điểm làm việc và cơ hội nghề nghiệp cho sinh viên/người mới đi làm.

## 2. Mục tiêu phân tích

- Phân tích tổng quan nhu cầu tuyển dụng IT tại Việt Nam.
- Phân tích kỹ năng và công nghệ được yêu cầu nhiều nhất.
- Phân tích mức lương theo vị trí, kinh nghiệm và địa điểm.
- Phân tích cơ hội nghề nghiệp IT cho sinh viên/người mới đi làm.

## 3. Cấu trúc thư mục

```text
data/           Dữ liệu gốc, dữ liệu đã xử lý và metadata
notebooks/      Notebook thu thập, xử lý và phân tích dữ liệu
dashboard/      Source code dashboard
ai_module/      Module tích hợp AI

reports/        Nội dung báo cáo và hình ảnh minh họa
